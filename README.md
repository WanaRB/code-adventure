# Code Adventure

Code Adventure adalah game 2D puzzle-platformer yang dibuat dengan Godot Engine.  
Kamu bermain sebagai Jikri, seorang karakter yang terjebak di dunia permainan dan harus melewati berbagai rintangan untuk mencapai pintu kemenangan di setiap level.

## Deskripsi

Lompati platform, hindari spike dan drone musuh, gunakan objek seperti laptop dan platform bergerak, lalu cari jalan menuju pintu keluar.  
Di beberapa titik, kamu akan menemui kuis yang menguji pemahamanmu terhadap permainan dan logika sederhana. Jawaban yang tepat akan membantumu membuka jalan atau mengubah kondisi level, sementara kesalahan bisa mengurangi darah.

Code Adventure dirancang untuk dimainkan langsung di browser (build web), dengan kontrol yang mendukung permainan di desktop dan sedang dikembangkan untuk pengalaman yang nyaman di perangkat mobile.

## Fitur Utama

- Game platformer 2D dengan elemen puzzle.
- Tiga level dengan tata letak rintangan dan musuh yang berbeda.
- Musuh berupa drone patroli dan spike yang harus dihindari.
- Objek interaktif seperti laptop, platform bergerak, dan pintu kemenangan.
- Sistem kuis di dalam game yang mempengaruhi progres level.
- Menu utama, pemilihan level, serta layar menang dan kalah.
